HTML

  • HTML is made up of elements that work together to provide content and the basic formwork for all pages on the web.
  • Content Elemnts: Text, Image, Header, List
  • Element types used to divide and describe a page: Meta-related elements, Content Division elements, Semantic elements.
  • The head element contains information about the webpage to help the browser render the page correctly. Not visible to users.
  • The body element represents the visible content shown to the user.
  • The footer element usually contains the author, copyright, contact, sitemap, and navigation.
  • HTML Attribute: defines additional information about an element. HTML attributes are alwyays defined in the beginning element, never the closing element.
  • The "style" element defines the CSS styles associated with a website.
  • The "title" element defines the title for a webpage.
  • The "base" element defines the base URL for a webpage.
  • The "link" element connects an external resource to the HTML document.
  • The "meta" element defines metadata such as the character set, description, keywords, author, and viewport.
  • The "script" element embeds client-side scripts in the HTML document.
  • The "charset" elements sets our character encoding.
  • The "http-equiv" element sets an HTTP header for content.
  • The "viewport" element sets the default view for our webpage.
  • Header elements range from "h1" to "h6" and should be used in order. The "h1" header should only be used once on each page.
  • Other common Body elements include: "p" for paragraph, "ul" for an unordered, or bulletpointed list, "ol" for an ordered list, and "li" for list items.
  • Image elements, "img", are used to display images and contain various attributes such as "src" defining the location of the image.
